# StackCalc Visualizer 🧮

A web-based tool designed to visualize the **Infix to Postfix conversion** and **Stack-based expression evaluation**. Built with a focus on educational clarity for Computer Engineering students.

## 🚀 Live Demo
Check out the live application here:
- **GitHub Pages:** [https://ahmedally00.github.io/project/]

## 🛠 Features
- **Visual Stack Operations:** Watch how elements are pushed and popped during calculation.
- **Step-by-Step Execution:** Breaks down complex mathematical expressions into stack operations.
- **Responsive UI:** Clean and modern interface built with **React** and **Vite**.

## 💻 Tech Stack
- **Frontend:** React.js
- **Bundler:** Vite
- **Styling:** CSS3
- **Logic:** Custom JavaScript Stack Implementation
